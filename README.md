# iconfont_dart

iconfont to dart.Icon can be implemented by calling iconfont classname.

## 使用

```
import 'package:iconfont_dart/iconfont_dart.dart';

main() {
  IconfontDart('../lib/assets/fonts/demo_index.html', './a.dart');
}
```

## 参数

* 第一个参数：demo_index.html所在的路径
* 第二个参数：生成的dart文件路径
